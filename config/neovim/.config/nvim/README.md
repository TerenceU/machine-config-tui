# Neovim Configuration

Custom Neovim configuration using **Packer.nvim** as plugin manager.

## Structure

```
.
├── init.lua                    # Entry point
├── lua/terence/
│   ├── init.lua               # Module loader
│   ├── packer.lua             # Plugin definitions
│   └── remap.lua              # Key remappings
├── after/plugin/
│   └── telescope.lua          # Telescope configuration
└── plugin/
    └── packer_compiled.lua    # Auto-generated by Packer
```

## Plugins

- **packer.nvim** - Plugin manager
- **telescope.nvim** - Fuzzy finder
- **plenary.nvim** - Lua utilities (telescope dependency)
- **telescope-fzf-native.nvim** - FZF sorter for telescope

## Key Mappings

**Leader key:** `Space`

### General
- `<leader>e` - Open file explorer (`:Ex`)

### Telescope
- `<leader>ff` - Find files
- `<leader>fg` - Find git files
- `<leader>fb` - List buffers
- `<leader>fG` - Live grep (search in files)

## Installation

This configuration is managed by `stow`. After running the installer:

1. Neovim config is symlinked from this directory to `~/.config/nvim`
2. Run `:PackerSync` in Neovim to install/update plugins
3. telescope-fzf-native requires `make` and a C compiler to build

## Plugin Manager

**Packer.nvim** is used instead of LazyVim. To manage plugins:

- `:PackerSync` - Install/update plugins
- `:PackerClean` - Remove unused plugins
- `:PackerCompile` - Regenerate compiled loader file
- `:PackerStatus` - Show plugin status
